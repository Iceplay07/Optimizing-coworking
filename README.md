Telegram бот для бронирования оборудования в коворкинге МАИ с интеллектуальной системой управления бронированиями. Поддерживает гибкое расписание, умное присоединение к существующим броням.


Основные функции

- Регистрация с проверкой статуса студента МАИ
- Бронирование на 4 недели вперед с учетом рабочих часов
- 3 типа бронирования: Лекторий, Плейстейшн, Компьютеры
- Умное присоединение к существующим бронированиям
- Просмотр и отмена своих бронирований
- Управление профилем (имя, телефон)
- Фильтр бронирований по датам и типам



Предварительные требования
- Python 3.9 или выше
- PostgreSQL 13 или выше
- Telegram Bot Token от [@BotFather](https://t.me/BotFather)
